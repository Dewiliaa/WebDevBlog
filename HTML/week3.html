🎉 Week 3! 🎉

Hi there!

In week 3, we delved into the practical aspects of web development by exploring the following key concepts:

1. Form Tag (form>/form>): We learned how to create forms using the form tag, which allows users to input data that can be submitted to a server for processing. Forms are fundamental for collecting user information or enabling user interaction on web pages.

2. Input Tag (input>): Within the form, we utilized the input tag to create various types of input fields such as text fields, checkboxes, radio buttons, and more. These input fields allow users to enter data into the form.

3. Label Tag (label>/label>): To associate labels with input fields, we employed the label tag. Labels improve accessibility and user experience by providing descriptive text for form elements. Clicking on a label focuses on the associated input field, enhancing usability.

4. Form Validation: We explored simple form validation techniques to ensure that the data submitted by users meets certain criteria or constraints. By implementing basic validation rules, such as requiring certain fields to be filled or validating input formats (e.g., email addresses), we can enhance data integrity and user experience.

Additionally, we continued to refine our understanding of responsive web design by applying media queries. Media queries allow us to adjust the layout and styling of our web pages based on the characteristics of the device or browser viewport, enabling a consistent user experience across various screen sizes.
